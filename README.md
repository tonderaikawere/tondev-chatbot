# Tondev AI Mentors

A powerful offline chat application that provides AI mentors specialized in various tech domains. Learn programming with intelligent, emotionally-aware AI mentors that understand your needs and guide you through your development journey.

## Features

- 🤖 7 Specialized AI Mentors
- 💡 Comprehensive Knowledge Base
- 🎯 Personalized Learning Experience
- 💬 Natural, Emotionally Intelligent Conversations
- 🔒 100% Offline Capable
- 📱 Modern, Responsive UI
- 🎨 Beautiful Design with Tailwind CSS

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tondev-ai-mentors.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
